# MPI による浅い部分木配布プロトタイプ

この文書は、リポジトリに追加した MPI プロトタイプの設計と使い方を簡潔にまとめたものです。

目的:

- root (rank 0) が深さ `D` まで浅い部分木を生成して、それらをラウンドロビンで各 rank に配布する
- 各 rank は担当サブツリーをローカルで列挙し結果を標準出力に出す（将来的に集約する想定）

使い方:

MPI 実行環境で次のように実行します:

```bash
mpirun -np 4 ./build/maximal_clique_mpi_bk graph.txt 2
```

注意:

- 現在の実装は単純化されたプロトタイプです。通信量や負荷均等化の改良が必要です。
- 実行には MPI C++ ライブラリが必要です（CMake で見つからない場合はビルドされません）。
